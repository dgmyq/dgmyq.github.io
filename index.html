前言：平时个人时不时会学一些东西，每次总是把学到的东西随便写在某个地方，导致下一次遇到同样的问题的时候又要去引擎上重新搜索一遍才能解决问题。
写博客这个想法很久就有了，只是自己懒得付出行动。写博客可以在很多平台上写，例如知乎，CSDN，简书，但是由于CSDN等软件过于流氓就不想在上面放自己的数据，所以有了自建博客的想法。

个人搭建博客的选择

上网搜到搭建个人博客有其中两种选择：

  • hexo + github
  • wordpress

wordpress可以安装很多插件，功能比较齐全。支持在线编辑发布。hexo比较简单，实现方法是在本地写好markdown文档然后在本地生成html再上传到github上面。

我搭建个人博客的目的只是为了记录自己学习的过程，不需要很多的功能，而且我现在用着自己的笔记软件obsidian，这个软件也是直接用md来记录，我可以先obsidian写好markdown然后再copy到hexo上也不费工夫，所以wordpress的在线编辑功能对我来说不是刚需，因此hexo看起来是比较合适的选择。

1.github篇

在github新建一个repository用于存放我们的网站，repository要做到下面这三点：

  • repository的名称要符合语法用户名.github.io，
  • 隐私勾选为Public，这样别人才可以看到你的博客
  • 勾选上Add a README file（看教程是这样做的，不知道为什么）

3.2 初始化hexo本地服务和测试

然后就要初始化我们的网站，输入hexo init初始化文件夹，接着输入npm install安装必备的组件。
